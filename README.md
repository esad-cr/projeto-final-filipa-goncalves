# projeto-final-filipa-goncalves
Projeto Final da UC - Laboratório de Projeto

Descrição
Para a realização do trabalho final para a cadeira Laboratório de Projeto II, irei realizar um site onde vou apresentar trabalhos desenhados realizados durante a licenciatura e trabalhos autónomos.


---

## Objetivos 
- Desenvolver uma página dentro do próprio portfólio;
- Organizar e apresentar trabalhos académicos;
- Aplicar os conhecimentos de HTML, CSS e JavaScript;
- Integrar projetos desenvolvidos em Processing. 

## Tecnologias Utilizadas 
-HTML5; 
-CSS3; 
-JavaScript;
-Processing.

---

## Páginas do Website

- **index.html** - Página inicial
- **sobre-mim.html** - Informação sobre o autor
- **projetos.html** - Apresentação dos trabalhos desenvolvidos 
- **contactos.html** - Contactos do autor
- **processing.html** - Projetos desenvolvidos em Processing (com canvas)
